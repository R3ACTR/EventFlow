EventFlow is a modular platform to manage hackathons and open-source programs efficiently.

## Features
- **Role-based dashboards**: Admin, Participant, Mentor, Judge
- **Event creation & management**
- **Team formation**
- **Project submissions**
- **Judge evaluation & scoring**
- **Announcements & notifications**
- **Certificate & badge generation**

## Tech Stack
- **Framework**: Next.js (App Router)
- **Language**: JavaScript
- **Backend**: Next.js Route Handlers
- **Database**: MongoDB (Mongoose)
- **Auth**: JWT / OAuth-ready
- **Styling**: Tailwind CSS
- **Package Manager**: pnpm

## Getting Started

1. Clone the repository
2. Install dependencies: `pnpm install`
3. Copy `.env.example` to `.env.local` and fill in the values
4. Run the development server: `pnpm dev`
